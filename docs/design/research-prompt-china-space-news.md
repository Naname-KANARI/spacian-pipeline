```
あなたは宇宙ニュースメディアの情報収集アーキテクトです。アイデア出しに協力してください。

# 背景
SPACiANという宇宙ニュースメディア（個人運営、月予算1万円程度、AI記者による自動記事生成パイプライン）
を運営しています。現在の収集ソースは英語サイト（SpaceNews, NASASpaceFlight, Breaking Defense,
Space.com）と日本語/JAXA英語版のRSSのみです。

中国は世界トップクラスの打上げ数を誇り、商業宇宙港（海南商業航天発射場、東方航天港など）の
動きも活発です。中国発の宇宙ニュースを見過ごすわけにはいかないと考えています。

# これまでの調査でわかった制約
- CNSA（国家航天局）、CASC、CALT、CASTなどの公式サイトはRSSを提供していない
- 中国航天報、我们的太空などの専門メディアはWeChat公式アカウント中心の発信で、Webサイトの更新は副次的
- WeChat公式アカウントをRSS化するサードパーティサービス（Wechat2RSS等）は存在するが、
  - 無料枠は収録アカウントが限定的
  - セルフホスト版はWeChatアカウントのログインが必要で、利用規約上のアカウント凍結リスクがある
  - 更新遅延が平均6時間程度
- 予算が少ないため、商用ニュースAPI（Reuters/APなど）の契約は難しい

# 相談したいこと
上記の制約を踏まえ、自動収集パイプラインで中国の宇宙ニュースを安定的に拾う方法について、
既存の発想（RSSHub・sitemapクローリング・WeChat2RSS）以外のアイデアがあれば教えてください。

例えば以下のような方向性も含めて、自由に発想してください:
- Google News等のニュース検索RSS（"site:"指定や中国語キーワード検索）を使う方法
- 中国の宇宙開発を専門的に英語で報じているジャーナリスト・ブログ・Xアカウントなど
  （独自の一次情報に近いものを英語で得られる経路があれば）
- 打上げスケジュール・実績を構造化データで提供しているサイト（NextSpaceflightやGunter's Space Page等）
- 低コストで使える翻訳・要約サービスとの組み合わせ
- その他、思いつく限りのアイデア

実現性（無料/低コストか、自動化しやすいか、更新の速さ）も含めてコメントしてもらえると助かります。
```
