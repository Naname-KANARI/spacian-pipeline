# CC設計相談: 多言語ニュースソースの拡大（v2: Gemini/ChatGPT調査結果統合）

作成: 2026-06-10（v2）

---

## 背景

現在の収集ソースは英語（SpaceNews / NASASpaceFlight / Breaking Defense / Space.com）と
JAXA英語版のみ（`config/sources.json`）。

ユーザーが中国語・ロシア語・ヒンディー語・アラビア語の現地語ソース追加を提案。
Gemini・ChatGPTにそれぞれ調査を依頼し、結果を突き合わせた（v1の簡易調査から更新）。

---

## 調査結果サマリー（両AI一致点を中心に）

### ロシア語 — 最有力候補
- **Roscosmos公式** (roscosmos.ru): RSS提供あり（`https://www.roscosmos.ru/rss/`、要実機確認）。毎日更新。英語版あり。著作権=B（政府機関・出典表示必須）。
- **TASS宇宙欄**: RSSあり（`tass.ru/rss/v2.xml`）だが全カテゴリ込みのため、宇宙カテゴリのフィルタリングが必要。著作権=D（通常メディア、本文スクレイピングはリスク）。
- 両AIとも「Tier 1・即導入候補」と評価。

### 中国語 — 価値は高いが技術的ハードル大
- CNSA/CASC/CALT/CAST いずれもRSS無し。**WeChat公式アカウント中心の発信**で、Webサイト更新は副次的。
- Sitemap経由のクローリングや、RSSHub等の非公式変換サービスが必要になりそう。
- 海南商業航天発射場・東方航天港（海陽）はWebサイト自体が未整備。
- 両AIとも「価値は最大級だが自動収集適性は低い」と評価（ChatGPT: Tier1、Gemini: Tier2＝技術的工夫が必要）。

### ヒンディー語 — 両AIとも見送り推奨
- ヒンディー語専門の宇宙メディアは実質存在しない。ISRO英語版＋一般ニュースのヒンディー語翻訳が実態。
- 既存の英語ソースでカバー可能なため、追加優先度は低い（Tier3）。

### アラビア語 — 「打上げ」軸とはズレる
- UAE Space Agencyはメディアセンター経由でRSS取得可能、著作権=B。
- ただしUAEは自国打上げ機を持たず、内容は衛星・宇宙探査・宇宙政策・国際協力が中心（打上げ情報はSpaceX等への相乗り文脈のみ）。
- Saudi Space AgencyはRSS無し、宇宙政策中心。

## CCへの質問（更新版）

### Q1: Roscosmos RSSの追加（最優先候補）
`https://www.roscosmos.ru/rss/`（要実機確認）を`config/sources.json`に追加するだけで、
既存の`collect.ts`アーキテクチャにそのまま乗りそうか。まずはこれ1件で技術検証したい。

### Q2: キリル文字タイトルのトークナイズ
score.tsの`tokenize`/`wordOverlap`はロシア語キリル文字に対して機能するか（単語分割自体は空白区切りで動きそうだが、英語ソースとの突合は意味をなさない）。
ロシア語ソース単独でのcandidate生成・generate.tsでの要約は問題なさそうか、CCの見立てを聞きたい。

### Q3: image_policy設定
Roscosmos=著作権B（政府機関・出典表示必須）の場合、`og_image_usage`は`"attribution_link_only"`が妥当か、それとも`"none"`から始めて様子を見るべきか。

### Q4: 中国語ソース（WeChat経由）は現実的か
RSS無し・WeChat中心という調査結果を受けて追加調査したところ、**Wechat2RSS** (wechat2rss.xlab.app) という
サードパーティサービスが見つかった。

- 無料枠: 一部の人気公众号をRSS化済み（GitHub issue経由で追加リクエスト可能、収録まで時間がかかる可能性）
- 有料/セルフホスト: `wewe-rss` / `we-mp-rss` 等のOSSあり。ただし**WeChatアカウントでのログインが必要**（私有部署）
- 更新遅延: 平均6時間程度（公称24時間以内）

「我们的太空」「中国航天報」がWechat2RSS無料枠に収録済みかは未確認。

懸念点:
- サードパーティサービス依存（単一障害点・継続性リスク）
- セルフホスト版はWeChatアカウントログインが必要 → 個人アカウントの規約リスクの可能性

これらを踏まえ、中国語ソースを「Tier2・要検証」として残すか、「将来課題」として今回見送るか、
CCの技術・リスク評価を聞きたい。

### Q5: ヒンディー語・アラビア語
両AIの推奨通り、ヒンディー語は見送り、アラビア語（UAE Space Agency）はTier2として保留としたいが、異論があれば。

### Q6: SNS（Telegram/WeChat）由来情報について
Geminiから「中国・ロシアのSDA関連動向はWeb記事よりTelegram/WeChatが先行する」という指摘があった。
これは現時点では別タスク・将来検討として切り離してよいか（パイプラインへの影響範囲が大きそうなため）。
