# 背景
SPACiANは宇宙ニュースメディアです（個人運営、AI記者による自動記事生成パイプライン）。
将来的には以下の情報基盤として利用します:
- AIによる自動記事生成
- 多言語情報収集
- 打上げ分析
- SDA（Space Domain Awareness）

現在は主に英語圏ソース（SpaceNews, NASASpaceFlight, Space.com, Breaking Defense）と
JAXA（日本語/英語）をRSSで収集しています。

今後、中国語・ロシア語・ヒンディー語・アラビア語圏の情報源を追加したいと考えています。

# 調査対象の分類

各言語について、以下を区別して調査してください。

① 一次情報源（最優先）: 宇宙機関、ロケット企業、衛星企業、発射場運営組織、政府機関
② 二次情報源: 宇宙専門メディア、科学技術ニュース、国営通信社

# 調査項目

各ソースについて以下を調査してください。

**基本情報**
- 名称
- URL
- 使用言語
- 運営主体

**自動収集適性**
- RSS有無・RSS URL
- Atom有無
- Sitemap有無
- JSON Feed有無
- robots.txt制限の有無

**更新頻度**: 毎日 / 週数回 / 月数回 など

**英語対応**: 英語版あり / 英語要約あり / 現地語のみ

**著作権分類**（以下から選択）
- A: 政府機関で再利用自由
- B: 政府機関だが出典表示必須
- C: 商用利用不可
- D: 通常ニュースサイト（個別確認要）
- E: 不明

**SPACiAN向け評価**（5段階評価）
- 打上げ情報の充実度
- 衛星情報の充実度
- 宇宙政策情報の充実度
- SDA（宇宙状況把握）的価値
- AI自動収集適性（RSS/構造化データの有無を反映）

# 言語別の特記事項

## 中国語
以下を必ず調査してください: CNSA（国家航天局）、CASC（中国航天科技集団）、
CALT（運載火箭技術研究院）、CAST（中国空間技術研究院）、中国航天報、
海南商業航天発射場、東方航天港（海陽）。
独立系の航天メディアがあれば併記してください。

## ロシア語
以下を必ず調査してください: Roscosmos公式サイト、Energia、Khrunichev、
Progress Rocket Space Centre、TASS宇宙欄。
RSSフィードの正確なURLが分かれば優先的に記載してください。

## ヒンディー語
ヒンディー語の専門宇宙メディアが実在するかを調査してください。
存在しない場合、または更新頻度が低い場合は「ISRO英語情報＋一般ニュースのヒンディー語翻訳が主流」など
現状を正直に説明してください。

## アラビア語
以下を調査してください: UAE Space Agency、Mohammed Bin Rashid Space Centre、
Saudi Space Agency。
UAEは自国の打上げ機を保有していないため、「打上げニュース」と「衛星・宇宙政策ニュース」の
どちらが中心になるかも説明してください。
